Jana Segelberg – Stínění s esencí (SPA, vícestránkový pocit)
=================================================
Hash routing (bez reloadu), React CDN + Tailwind – běží na GitHub Pages bez buildu.

Jak nasadit:
1) Nahrajte všechny soubory do kořene repozitáře.
2) Settings → Pages → Deploy from a branch → main (root).
3) Otevřete URL. Navigace funguje přes hash: #/about, #/gallery atd.

Hotové podle zadání:
- světlo zleva, rychlé přechody (0.5 s), hover underline v menu, hover CTA,
- fade-in hero, menší hero na podstránkách, dvojjazyčnost CZ/EN (paměť jazyka),
- responzivní layout, lightbox v galerii, jednotná hlavička/patka,
- Kontakt zatím bez odesílání (připomenout napojení na Google Sheets po schválení).
